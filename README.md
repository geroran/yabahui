# 模拟举办哑巴会 · 页面概览

本仓库为微信小程序项目，主题是彝族火把节期间“哑巴会”的仪式体验。项目使用标准小程序结构（`app.js/json/wxss`、`pages/`、`components/`），可直接导入微信开发者工具运行。

## 页面导航

| 页面 | 路径 | 说明 |
| --- | --- | --- |
| 首页 | `pages/index/index` | 仪式介绍与关卡入口。展示活动背景、角色说明，并引导进入两个互动小游戏。包含全局返程导航。 |
| 祭拜神树 (2-1) | `pages/2-1/index` | 记忆顺序游戏。玩家需按照神树提示的顺序执行“点燃香火”“敬献酒水”等步骤。包含倒计时、演示阶段、成功/失败弹窗与重试按钮。 |
| 恭请大哑巴 (2-2) | `pages/2-2/index` | 装扮仪式游戏。按照正确顺序为“大哑巴”穿戴龙纹彩绘、牛皮头饰、羽毛、法器。包含逐项动画、错误震动提示以及完成后的文化介绍。 |

## 组件

- `components/navigation-bar/`：自定义顶部导航，提供标题、背景与可复用的返回逻辑。各页面也可使用 `back-btn` 配合 `goBack` 方法返回上一页或首页。

## 运行与调试

1. 在微信开发者工具中选择“导入项目”，目录指向仓库根目录，`appId` 使用个人/测试号。
2. 保持 `project.config.json` 为默认配置即可进入模拟器预览。
3. 需要真机体验时，使用开发者工具的“预览”功能生成二维码扫描运行。

## 目录速览

```
app.js / app.json / app.wxss   # 全局入口与样式
pages/                         # 所有页面，子目录含 index.js/json/wxml/wxss
components/navigation-bar/     # 自定义组件
project.config.json            # 微信DevTools工程配置
```
